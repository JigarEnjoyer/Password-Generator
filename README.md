# Password-Generator
A simple-to-use, local, and open-sourced password generator built using Python. 


<img width="317" height="414" alt="Screenshot 2026-01-14 161026" src="https://github.com/user-attachments/assets/59e8afde-b78e-4f04-b52e-f53bdb0c10eb" />

## Features:
- Create random passwords from 4 to 64 characters (Default is 12)
- Add complexity by adding symbols, numbers, and upper/lower characters
- Exclude certain symbols that are not supported widely for passwords: {}}[]()/\'\"`~,;:.<>
- Small file size (~12KB)

## Installation:

Go to [Releases](https://github.com/JigarEnjoyer/Password-Generator/releases) and download "Password Generator.exe"


## Run from source:

# 1. Clone repository
    git clone https://github.com/JigarEnjoyer/Password-Generator.git
    cd Password-Generator

# 2. Install dependency
     pip install -r requirements.txt

# 3. Run the app
     python Main.py
